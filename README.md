# WordCounter

Simple Java application for producing a word and frequency count for a given text sample, amongst other things.

### Installing

Clone the git repository and run using gradle:

```
./gradlew build
./gradlew run
```

## Running the tests

Run the unit tests by running the test task
```
./gradlew test
```
